# PyMessageChat

## Install application:

  ### After download:
   $ cd PyMessageChat/app/code

   $ sudo apt install python3-venv

   $ python3 -m venv env

   $ source env/bin/active

   $ pip3 install -r requirements.txt

## Run app:
  $ cd PyMessageChat/app/code
  
  $ source env/bin/active

  $ python3 app.py

## Run webbrowser on 127.0.0.1:5000/
